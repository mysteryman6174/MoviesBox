Welcome to MoviesBox

Movies are a popular form of entertainment all around the world. They offer a way for people to escape from their daily lives and immerse themselves in different stories, characters, and worlds. There are many different genres of movies, including action, comedy, drama, horror, romance, and science fiction. Each genre appeals to different audiences and can evoke a wide range of emotions.

When it comes to choosing a movie to watch, there are many factors to consider. Some people prefer to watch new releases, while others enjoy revisiting classic films. It's also important to think about the rating of a movie and whether it's appropriate for the audience. For example, parents may want to avoid showing their young children movies with violence or adult themes.

This website is a personal project for me to learn and improve my knowledge of HTML, CSS Bootstrap, API knowledge,
and the Flask Framework in Python in order to create a dynamic website.

The website is built with Python's Flask Framework, it requests information from the TMDB Movies API and displays it with HTML and Bootstrap CSS, Some of the features are as follows:

  • Search Functionality
  • Discover movies using various parameters such as popularity, release date, average ratings, etc.
  • Access to all movie information such as synopsis, cast information, blockbuster revenue, runtime, etc.
